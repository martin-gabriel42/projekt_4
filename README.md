#Projekt z SQL - ENGETO

Tento repozitář obsahuje skripty a doprovodnou dokumentaci sloužící k vytvoření podkladů pro zodpovězení výzkumných otázek.

##Obsah repozitáře

###1. martin_gabriel_table_generation_script.sql
     
SQL skript, který vytváří podkladová data ve formě dvou tabulek v databázi data_academy_2025_06_11.

###3. martin_gabriel_query_script.sql

SQL skript obsahující dotazy, pomocí kterých lze přímo odpovědět na výzkumné otázky z vytvořených tabulek.

 ###3. doc.md

Textový soubor obsahující:
- popis tvorby tabulek
- popis tvorby dotazů
- interpretaci dat a finální odpovědi na výzkumné otázky
